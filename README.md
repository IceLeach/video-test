# 视频测试

## 使用方式

安装依赖

```bash
$ yarn
```

启动项目

```bash
$ yarn start
$ yarn start:test
$ yarn start:pro
```

打包部署

```bash
$ yarn build
$ yarn build:test
$ yarn build:pro
```
